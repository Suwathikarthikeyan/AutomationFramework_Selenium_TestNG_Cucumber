Automation Testing Framework – Selenium + TestNG

This repository contains a scalable and maintainable automation testing framework built using Selenium WebDriver, Java, and TestNG.
The framework is designed to support cross-browser testing, data-driven execution, and test suite management through testng.xml.

Key Highlights
Selenium WebDriver with Java
TestNG for test execution and suite configuration
testng.xml for parallel execution and parameterization
Cross-browser support (Chrome / Firefox)
Page Object Model (POM) for clean architecture
Reusable utilities for browser setup and waits
Maven for dependency management
Structured reporting and clear test logs

Project Structure
src/test/java
 ├── base        → Browser & test setup
 ├── pages       → Page Object classes
 ├── tests       → TestNG test cases
 └── utils       → Common utilities
testng.xml       → Test suite configuration
pom.xml          → Maven dependencies

How to Run

Clone the repository
Open the project in IntelliJ / Eclipse
Update browser or URL in testng.xml
Run testng.xml or individual test classes

Objective

To demonstrate real-world automation testing practices aligned with industry standards and job-ready expectations
